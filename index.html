Enhanced MiNote

Enhanced MiNote MiNote is a simple yet functional Python application that allows users to write personal notes, edit them, and create structured “To-Do Lists” (Activities Lists) to help increase productivity and track goals. The application is designed to make note-taking quick and easy through a clear and minimal graphical interface built with Tkinter, which runs entirely in Python without any external libraries. This project was created as part of a learning project in Python, demonstrating how much can be done using only Python’s standard features. All data is stored persistently in plain text files inside the MiNote/ folder, ensuring that users’ notes and to-do lists are saved and accessible across sessions.
